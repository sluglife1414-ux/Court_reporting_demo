## TRANSCRIPT STRUCTURE

### Caption Page (Page 1)
- Running header: WCB case number (e.g., `WCB G3953702`) centered at top
- Court header:
  ```
  WORKERS' COMPENSATION BOARD
  STATE OF NEW YORK
  ```
- Caption border: dashes + "x" on each side (NY WC format: `------ x`)
- Party format:
  ```
  [CLAIMANT NAME], Claimant,
  -against-
  [EMPLOYER NAME], Employer.
  ```
- Sub-header block:
  ```
  WCB Case No.:     [number]
  Carrier Case No.: [number]
  D/A:              [date of accident]
  ```
- Depo type line:
  `TELEPHONIC DEPOSITION OF [WITNESS FULL NAME], a non-party witness herein...`
- Caption closes with asterisks: `*** ... ***`

### Appearances Page
- Header: `A P P E A R A N C E S:`
- Attorney blocks:
  - Attorneys for Claimant (no firm address if not provided)
  - `BY: [ATTY NAME], ESQ.`
  - Attorneys for Respondent
  - `BY: [ATTY NAME], ESQ.`
- Terminator: `xxxxx` (standard NY appearances end marker)
- No videographer block (telephonic depo)

### Oath / Examination Start
- Flows directly into testimony (oath lines stitch onto top of first testimony page)
- Format:
  ```
  [WITNESS SPACED NAME],
  Having been first duly affirmed before a Notary
  Public of the State of New York, was examined
  and testified as follows:
  EXAMINATION BY
  MR. [ATTY LAST]:
  ```
- Testimony Q/A starts immediately after on same page

### Testimony Pages
- Q/A format: `Q.  [text]` / `A.  [text]` (2 spaces after period)
- Q. and A. labels: bold in PDF, regular in plain text
- Colloquy (MR./MS./THE WITNESS:): indented, no bold
- Interruptions: double dash (--) — NOT em dash. Verified from Wade finished transcript (KB-NY-002).
- Trailing off: ellipsis (...)
- Off-record: `(Off the record.)` indented
- Running header on every testimony page: WCB case number centered

### Witness Certificate
- Reporter certifies as Notary Public, State of New York
- Uses NY notary language (NOT Louisiana CCR language)
- No CCR/RPR designation

### Index Page
- After witness certificate
- Tabbed format: WITNESS tab PAGE tab

- Present in AD's Fourman output

## OBJECTION FORMAT (NY WC)

NY WC objection format differs from Louisiana:

| Situation               | Correct Format                    |
|-------------------------|-----------------------------------|
| Single objection        | `Objection.`                      |
| Objection to form       | `Objection to form.`              |
| Multiple bases          | `Objection to form; foundation.`  |
| Foundation only         | `Objection, foundation.`          |

DO NOT use Louisiana format (`Objection to form.` with semicolon separation is correct,
but `Objection, form.` is the Louisiana pattern — do not apply here).

## Q/A TOGGLE BEHAVIOR

- First speaker after oath: **Q.** (examining attorney)
- Alternates Q/A for unlabeled blocks
- Labeled colloquy (`MR. BLUM:`, `MR. FRIEDLICH:`) does NOT consume a toggle slot
- `[REVIEW:...]` tags at start of a block are a boundary trigger (toggle-safe)
- AD depos are telephonic — no videographer block

## KNOWN STATE-SPECIFIC RULES

1. WCB depositions are telephonic — no in-person location block required.
2. WCB case number (`WCB G#######`) appears in every running header.
3. "WORKERS' COMPENSATION BOARD" is all caps in the caption.
4. "STATE OF NEW YORK" is all caps in the caption.
5. Claimant/Employer are the party designations (not Plaintiff/Defendant).
6. Date of accident (`D/A`) is a required caption field.
