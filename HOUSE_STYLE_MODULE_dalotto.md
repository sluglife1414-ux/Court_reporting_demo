## TRANSCRIPT STRUCTURE (NY WC)

### Caption Page (page 1)
- Running header: WCB case number (e.g., `WCB G3953702`) centered at top
- Caption block:
  - Court: WORKERS' COMPENSATION BOARD / STATE OF NEW YORK
  - Border: dashes + "x" on each line (NY WC style: `------ x`)
  - Party format: `JAMES [CLAIMANT], Claimant, -against- [EMPLOYER], Employer.`
  - WCB Case No. + Carrier Case No. + D/A (date of accident) in caption
- Depo type line: `TELEPHONIC DEPOSITION OF [WITNESS NAME], a non-party witness herein...`
- Caption closes with asterisks: `*** ... ***`

### Page 2 — Reporter name only
- AD's name appears alone on page 2 (observed in Fourman output)
- Likely carries over from caption formatting — confirm with sample depo

### Appearances Page (page 3)
- Header: `A P P E A R A N C E S:`
- Attorney blocks:
  - Attorneys for Claimant (no firm address if not provided)
  - BY: [ATTY NAME], ESQ.
  - Attorneys for Respondent
  - BY: [ATTY NAME], ESQ.
- Terminator: `xxxxx` (standard NY appearances end marker)
- No videographer block (telephonic depo)

### Oath / Examination Start (page 4)
- Flows directly into testimony (oath lines stitch onto top of first testimony page)
- Format:
  ```
  [WITNESS SPACED NAME],
  Having been first duly sworn before a Notary
  Public of the State of New York, was examined
  and testified as follows:
  EXAMINATION BY
  MR. [ATTY LAST]:
  ```
- Testimony Q/A starts immediately after on same page

### Testimony Pages
- Q/A format: `Q.  [text]` / `A.  [text]` (2 spaces after period)
- Q. and A. labels: **bold** in PDF, regular in plain text
- Colloquy (MR./MS./THE WITNESS:): indented, no bold
- Interruptions: em dash (—)
- Trailing off: ellipsis (...)
- Off-record: `(Off the record.)` indented
- Running header on every testimony page: WCB case number centered

### Witness Certificate (page 21 in Fourman — 28-page depo)
- `I, ALICIA D'ALOTTO, hereby certify that the`
- NY notary certificate language (not Louisiana CCR language)
- AD signs as Notary Public of the State of New York

### Index Page
